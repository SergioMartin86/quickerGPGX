quickerGPGX
--------------

[![Build & Tests](https://github.com/SergioMartin86/quickerSnes9x/actions/workflows/make.yml/badge.svg)](https://github.com/SergioMartin86/quickerSnes9X/actions/workflows/make.yml)


quickerGPGX is an attempt to optimizing [Genesis Plus GX (gpgx)](https://github.com/ekeeke/Genesis-Plus-GX). The main aim is to improve the performance of headless re-recording for TASing and botting (See: [JaffarPlus](https://github.com/SergioMartin86/jaffarPlus)) purposes. However, if this work can help regular play emulation, then much better.
